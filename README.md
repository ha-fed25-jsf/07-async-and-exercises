# Övningar
## Routing, state m.m.

### Skapa nytt repo
```bash
 npm create vite@latest
 # Välj JavaScript + React compiler

 cd projekt-namn
 npm install react-router zustand
 git init
 code .

 # Gärna i separat terminal
 npm run dev
```

### Lägg till routing
1. main.jsx - lägg till RouterProvider
2. routes.js - lista med route-objekt
3. pages/Root.jsx, Root.css - root layout
